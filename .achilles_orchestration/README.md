# Romer-MPL ACHILLES Orchestration

This folder seeds the non-destructive orchestration layer for the Romer-MPL repo.

## Role

Romer-MPL owns Maximum Probable Loss modelling, risk/disposition modelling, compact analysis outputs, and mission/GMAT-support exports where required.

## Rule

Large result sets should be compacted into ranges, dispositions, risk bands, recommended next configurations and Sheets-ready outputs.

No existing file should be deleted or overwritten during scaffold installation.
